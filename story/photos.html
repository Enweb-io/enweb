ANATOMY OF THE CODE

First we create a div and give it a class card inside card we create an image hosting container and give it card-image. The image will live inside this container but we have to provide it as a link that is why we have placed it within the anchor tags. This turns the image into a clickable link and we use href="swimsuit/9.jpg" to open the full image. So when the user clicks on the thumbnail the large image will open. However we need to tell the browser that we are using fancybox so we add this line to the anchor tag. data-fancybox="gallery" data-caption="Code Academy these two lines work together as one opens the image modal and the other adds a caption to the opened image. That is all there is to it. Now let us style the card and the parent container.

  align-items: center;

GRID LAYOUTS

We are going to use Grid Layouts to create a three column layout on Desktop and we will use queries to adjust to two columns on small devices. In his design we do things differently we will not float the columns or cards we infact have defined with the parent divs how the cards should be displayed using grid-template-columns: repeat(3, 1fr) and grid-gap: 1rem; where 3 is the number of columns and 1fr is used to insert object fit so that all the images have the same dimensions and the content will properly fit without stretch or distortion. This means that images of different sizes will be automatically cropped so that all images have a uniform height this is good for a smart and neat layout. Grid gap creates spaces between the images. Set this to 0 to close the gaps. Proceed with the rest of the CSS.

SWITCH TO 2 COLUMNS ON SMALL SCREENS

On smaller screens we need two columns and we use @media and max-width to implement the change. The following code translates to if the screen width is >= 800 then switch to two columns. 

	                   
@media only screen and (max-width: 800px) {
  .main .container {
    display: grid;
     grid-template-columns: repeat(2, 1fr);
    grid-gap: 1rem;
  }
}
