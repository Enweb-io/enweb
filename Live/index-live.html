HOW TO CREATE IMAGE COLUMNS

To build an image column or Place Images Side by Side we first insert the images that will make up the Image span. But first create a work station. Create a folder named Live inside Live place an images folder and put any three images of the same dimension in it. We will start with the following basic HTML document. Type it in Notepad++ then go to file and save the file as index.html in the live folder. To run the project go to the run button then select Launch in Chrome. For a better interface. INSERT THREE IMAGES

To insert an image we use the image tag with the following syntax 


  <img src="productimages/cat.jpg" alt="img" >

Img is the name of the tag src refers to the source of the image in our case our source is the images folder inside the images folder is an image named cat with file extension .jpg we use alt to define what should display in the browser if the image is missing. Let us insert three images


  <img src="productimages/1.jpg" alt="img" >
  <img src="productimages/2.jpg" alt="img" >
  <img src="productimages/3.jpg" alt="img" >

PLACE IMAGES IN THE BODY TAG

The body element hosts what is displayed in a web browser. To view the images we need to place the image code in body element as shown by the following code. 


<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>IMAGES</title>
<style>
</style>
</head>
<body>
  <img src="productimages/1.jpg" alt="img" >
  <img src="productimages/2.jpg" alt="img" >
  <img src="productimages/3.jpg" alt="img" >
</body>
</html>

STYLE THE IMAGES

To create an image span we need to divide the width of the screen by the number of images we want to display side by side. The width is 100% and we want three images therefore 100/3 = 33.33 so each image will be given a width of 33.33% some padding to create spaces between the images and finally floated to the left to make them display side by side. Then we add box-sizing to prevent the added padding from altering the set widths. We can put this data into the following CSS.


<style>
* {
  box-sizing: border-box;
}

img {
  width: 33.33%;
  float:left;
  padding: 5px;
}
</style>

FINAL CODE

TWO COLUMN LAYOUT

Notice how the images are too small on mobile if it was your ex-girlfriend's wedding photo you would not even see her face. ARISING COMPLICATIONS

Student

What we have done so far is good but we may have complications in future for example if we want to add an image caption and a buy button we would need a lot of work because each image does not exist within its own container. If we have an image in its own container then we can place other things in there like a button or image caption therefore let us add each image to its own parent container. We will create a container named column and we will place each image in its column with the following code.


<div class="row">
  <div class="column">
    <img src="productimages/1.jpg" alt="" >
  </div>
  <div class="column">
    <img src="productimages/2.jpg" alt="" >
  </div>
</div>

APPLY CSS TO THE COLUMN

So this time we do not apply the CSS to the image we apply it to the column but the width of the image within its container must be 100% and we use the following code


* {
  box-sizing: border-box;
}

img {
  width:100%;
 } 

.column {
  width: 50%;
  float:left;
  padding: 5px;
}
